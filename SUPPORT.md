# Support

Use GitHub Discussions for setup questions and ideas, and GitHub Issues for
reproducible bugs and scoped feature requests. Use private security reporting
for vulnerabilities; see [SECURITY.md](SECURITY.md).

When requesting help, include the Adaetum version, installer/OS, relevant
`platform.yaml` settings, sanitized logs, and the exact validation command that
failed. Do not share secrets, tokens, private hostnames, or recovery artifacts.
